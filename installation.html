Démarrer avec ZenStyle

Installer le framework

ZenStyle fonctionne avec une simple feuille de style. Aucun outil de build n'est obligatoire.

1. Copier les fichiers

Téléchargez les sources lisibles ou les fichiers minifiés prêts pour la production.

Télécharger le CSSCSS minifié

2. Ajouter les liens

<link rel="stylesheet" href="zenstyle.css">
<script src="zenstyle.js" defer></script>

3. Utiliser un composant

<button class="zs-btn zs-btn-primary">
  Commencer
</button>

Voir tous les composants

Installation avec npm

npm install zenstyle

Importez ensuite zenstyle/zenstyle.css et, pour les composants interactifs, zenstyle/zenstyle.js.
